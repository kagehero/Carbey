# 機能実装状況一覧

## 現在実装済み（Implemented）

### MVP / 基盤機能
| 機能 | 説明 |
|------|------|
| 在庫管理 | 在庫一覧・詳細・編集 |
| カーセンサー連携 | Python同期スクリプト、API同期 |
| 定期同期（1日2回） | Vercel Cron: 9時・18時(JST)。CRON_SECRET必須。vercel.json参照 |
| 一覧・検索・ソート | 在庫テーブル、メーカー別フィルタ、ソート（価格・滞留・CVR・AI推奨割引率など） |
| 滞留日数 | 計算・表示・バンド別分類 |
| 基本指標 | CVR計算、閲覧数・問合せ数表示 |
| シンプルなダッシュボード | KPIカード、本日の確認事項（アラート） |

### ダッシュボード・グラフ可視化
| 機能 | 説明 |
|------|------|
| 滞留分析グラフ | 不良在庫割合（線）＋滞留台数（棒）の複合グラフ |
| CVR分析グラフ | 要改善・改善見込み・優秀の棒グラフ比較 |
| 価格最適化グラフ | 値下げ対象台数（棒）＋割合（線）の複合グラフ |
| 滞留日数分布 | バンド別台数の棒グラフ |
| CVR統計 | 平均CVR、優秀/良好/要改善の内訳 |
| 優先販売ランキング | 滞留＋CVRスコアによるテーブル |
| 値下げ検討対象 | 滞留60日以上 or CVR 2%未満の一覧 |

### AI分析・将来予測
| 機能 | 説明 |
|------|------|
| AI分析ページ | 専用ページ（/admin/analytics/ai） |
| CVR改善シミュレーション | 要改善→2.5%想定時の想定追加問合せ・販売台数 |
| 想定回転率 | 現状 vs 改善後の月間回転率比較 |
| 想定売上高 | 現状 vs 改善後の月間売上、想定追加売上 |
| AI分析タグ | サイドバーに「AI」バッジ表示 |

| 滞留分析 | バンド別詳細テーブル |
| CVR分析 | 優秀/良好/要改善の分類表示 |
| 価格最適化 | 最適化・AI提案の両方、一括反映 |

### 価格・バルク操作
| 機能 | 説明 |
|------|------|
