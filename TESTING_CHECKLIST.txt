COMMON ISSUES & SOLUTIONS
================================================================================

ISSUE: "Could not find the table 'public.inventories'"
SOLUTION: Run database migrations in Supabase SQL Editor

ISSUE: "Login failed - Invalid credentials"
SOLUTION: 
1. Verify user exists in Supabase Auth
2. Check email/password are correct
3. Look for errors in browser console

ISSUE: Dashboard shows "0" for all KPIs
SOLUTION:
1. Run: python carbey.py sync
2. Check Supabase: select count(*) from inventories;
3. Verify data synced successfully

ISSUE: "Access denied" or blank pages
SOLUTION:
1. Check user_profiles table has entry for your user
2. Verify role = 'admin'
3. Run: select * from user_profiles where id = auth.uid();

ISSUE: Charts not displaying
SOLUTION:
1. Check browser console for errors
2. Verify recharts is installed: npm list recharts
3. Check if data is loading: F12 → Network tab

ISSUE: Search/filter not working
SOLUTION:
1. Check browser console for errors
2. Verify data is loaded
3. Try hard refresh: Ctrl+Shift+R

ISSUE: Sidebar menu items show 404
SOLUTION: This is expected for Phase 3 features (analytics pages, edit pages)
          Only Dashboard and Inventory Management work in Phase 2
